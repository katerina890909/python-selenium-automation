#   Автоматизация тестирования (Python + Selenium)
Мой первый проект по автоматизации веб-интерфейса.
##  Что делает этот проект:
* Автоматизирует процесс регистрации / входа на сайте.
* Проверяет корректность ввода данных.

##  Стек технологий:
* Язык:Python 3
* Библиотека: Selenium WebDriver
* Инструменты: VS Code, Git
